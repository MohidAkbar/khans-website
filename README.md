# 🚖 TAXI RAYAN ARNHEM - Taxi Booking Website

Professional taxi booking website for TAXI RAYAN ARNHEM, serving Arnhem and surrounding areas in the Netherlands.

## 🌐 Live Website
**[www.taxirayanarnhem.nl](https://www.taxirayanarnhem.nl)**

---

## ✨ Features

- 📱 **Fully Responsive** - Works on mobile, tablet & desktop
- 🌍 **Bilingual** - Dutch (NL) & English (EN) language toggle
- 💬 **WhatsApp Booking** - Direct booking via WhatsApp
- 🚗 **Car Fleet Showcase** - Volvo V90, Mercedes E-Klasse, Tesla Model S
- 📍 **Service Area** - 25 locations in and around Arnhem
- ⭐ **Customer Reviews** - Social proof section
- 📝 **Complaint Box** - Customer feedback form
- 🎨 **Modern Design** - Dark theme with gold accents
- ✨ **Smooth Animations** - Scroll reveal effects
- 🔒 **Secure** - HTTPS enabled via GitHub Pages
